Submission: Katalog Restoran pada kelas Menjadi Front-End Web Developer Expert

## Run Locally
Clone the project
```bash
  git clone https://github.com/takasicode/eatspace-v1
```

Go to the project directory
```bash
  cd eatspace-v1
```

Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm run start-dev
```
